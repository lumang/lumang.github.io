raspberry login

2018-05-13

现在树莓派作为微型卡片电脑，越来越多的应用到各种产品和创意设计中，树莓派在实际使用通常是无显示器，通过ssh远程操作那么问题来了
在没有显示器的情况下如何获取raspberry树莓派的IP地址呢？
一般的话有三种解决方法

1、将树莓派和路由器用网线连好以后，到路由器页面上去查看树莓派的IP地址

这种方法比较简单，一目了然，但是对于没有路由器管理员权限的情况下比较难实现，特别是在校的学生和在外租房子的同学们

2、通过arp -a 命令

window环境下将树莓派和计算机网口通过网线连接，且在计算机网口设置固定IP地址
这样通过cmd 窗口输入 arp -a 命令后可以查看到树莓派的IP地址，这个比上面路由器的方案相比就比较简单了，但是还有有问题，因为arp 缓存问题，有的时候根本无法查到树莓派IP地址，这就引出了第三种方式了。通过网络调试神奇wireshark软件

3、通过Wireshark软件来获取树莓派的IP地址

这里不需要了解太多Wireshark的原理,只需要了解简单网络基本常识即可，具体来看我解决问题的步骤

  • 首先跟第二种方法中的类似将树莓派和计算机连接并把计算机网卡设置为固定IP地址
  • 然后打开wireshark软件选择捕获接口为本地链接，软件开始捕获计算机网口和外界的通信数据了
  • 将树莓派通电，通电以后树莓派上就会显示一系列通信数据，大概1-2分钟后查看是否有Raspber开头的MAC地址如图，可以看到send MAC address 和 send IP address
  • 将计算机的IP地址改为和raspberry 相同网段的地址就可以登录到raspberry了
