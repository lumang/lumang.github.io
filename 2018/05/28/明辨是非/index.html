明辨是非

发表于 2018-05-28
字数统计: 810 | 阅读时长 ≈ 3

最近有几则新闻先跟大家分享下

失控宝马车检查测试后，没有问题
腾旭投资“差评”，涉嫌洗稿，“差评”官方道歉
河南小女孩不凶去世，网上因为女孩家人众筹，怀疑捐款被挪用，微博大V带头实名声讨

以上这些事情，在上周可以说是舆论的风口浪尖了，具体事情内部我这里就不详细描述了，有兴趣的读者可以，到网上搜索下，了解事情的经过，来龙去脉，但是对于网上的信息的甄别就是我们今天要讨论的了。

现在是一个信息爆炸的时代，每天互联网上产生的数据信息，比过去几千年的综合数据还要多，这么多信息，通过手机，电脑，互联网，广播电视等，源源不断的，进入我们的眼里，耳朵里，头脑里，我们大脑充满了各种信息。那么这些信息肯定不是所有我们都需要，也不是所有信息都对与个人是有价值的，这里就需要我们根据信息来甄别取舍，这里重点说下如何判断事情的真伪。

新闻记者追求事件的真实性，但是自媒体时代，很多普通人成为了所谓自媒体，他们为了点击率，粉丝量，吸引眼球，估计断章取义，在没有完全掌握事实根据的情况下，肆意弯曲传播事件，在加上社交网络的发的，普通人的一次小小的转发举动，获取就引起了轩然大波。

作为我们普通人，吃瓜群众，如何不被这些事情所迷惑需要我们学习，首先对于社会热点新闻一点要谨慎传播评论，对于没有权威媒体报道的事情，一定不要轻易的传播，特别是对当事人有明确伤害，侮辱的事情，这次河南小姑娘的家人，由于网络大V的推波助澜，在没有事实根据的情况下，发表一些不合理的言论，被广大不明真相的网友肆意的批判，平白无故的受到了二次伤害。这个值得每一个人警惕。

法律上将捉贼捉赃，凡事讲究的是证据，如果没有震惊，眼见不一定为实，证据还需要相互印证才有效，法律绝不是信口开河，很多网络上的网友遇到事情，张口就来，很多人没有经过大脑就发出声音，对事件的发展起到了推波助澜的作用。这种事情在生活正屡见不鲜，很多朋友圈的求助信息，谣言就是这样传播开了的。

综上如何在网络明辨十分，判断真假
1、了解事情来龙去脉，不妄下结论，在不了解情况下更不能传播
2、收集资料，了解事情发生的过程，不要只关注结果
3、关注权威媒体的发布信息
4、遇事情三思而后行，不要为一时的口快
